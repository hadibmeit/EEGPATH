EEGPATH
EEGPATH is a lightweight macOS application built with SwiftUI that allows users to load EEG data from CSV files and visualize the signal as a line chart.
The tool provides a simple, fast, and intuitive way to inspect EEG waveforms without relying on heavy desktop platforms such as MATLAB or LabChart.
